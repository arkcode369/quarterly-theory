# 02 - Deeper Dive into Quarterly Theory

## Konsep yang Diajarkan
1. Cara identifikasi consolidation di Quarter 1
2. Timeframe yang benar untuk melihat true openings
3. Hari-hari yang tidak boleh di-trade

## Rangkuman

### Hari/Event yang Tidak Boleh Di-Trade
1. Core CPI y/y
2. CPI m/m
3. PPI m/m
4. Core PPI
5. FOMC
6. Setelah holidays
7. NFP
8. Setelah large range
9. Rate announcements
10. Unemployment claims

> **Catatan**: Bisa trade di hari tersebut, tapi hanya **setelah** event selesai. Jika large range terjadi di satu session, trade di session berikutnya.

### Timeframe untuk True Openings
| Cycle | View pada TF |
|-------|-------------|
| Yearly true opening | Weekly |
| Monthly true opening | Daily |
| Weekly true opening | 4 Hour |
| Daily true opening | 1 Hour |
| True session opening (90min) | 5 Minute |

### Q1 pada Quarterly Theory
- **Abaikan Q1** — hanya fokus pada Q2 dan Q3
- Cari consolidation di dalam Q2 dan Q3
- Konsep ini berlaku di **semua timeframe**

### Cara Aplikasi di Chart
1. Setup Q1 pada 90-minute cycle
2. Drop ke 5-minute timeframe, setup micro cycle
3. Abaikan Q1, cari consolidation di Q2 dan Q3

## Rangkuman

### Precision Swing Points - Informasi Baru
- Precision swing points **bisa overlap** satu sama lain
- Jika original precision swing point memenuhi requirements untuk sebuah precision swing point baru, tapi new precision swing point tidak → tetap valid
- Jika precision swing point memiliki expansion karena menjadi bagian dari swing point → bisa dihitung sebagai MSS? Ya
- Jika trend bearish → true cycle opening harus di atas, BPR, precision swing point harus di dalam precision range

### Tujuan Precision Swing Point
- Untuk menguji apa yang terjadi ketika precision swing point align dengan precision swing point lain
- Jika semua bearish → valid

### New Trading Model
**Requirements:**
1. Breakaway gap
2. FVG
3. True cycle opening
4. 2 precision swing points
5. SSMT
6. High-impact news event

**Setup untuk bearish trend:**
1. Define high dari high-impact news event candlestick (expanded)
2. Cari SSMT di atas high impact news event candlestick
3. Price action expands lower, lalu retraces ke atas
4. Candlestick retracement yang retrace ke precision swing point's low → membuat FVG dengan candlestick lain
5. Price action masuk ke FVG lalu expands → entry di sini
6. Expect breakaway gap

### Low Probability Price Action
- Ketika HTF dan LTF tidak align
- Contoh: jika SSMT menunjukkan bullish tapi QT alignment (Weekly + Monthly di dalam cycle yang sama) menunjukkan bearish → low probability
